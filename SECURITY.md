# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you have found a potential security vulnerability, please do **not** open a public issue.
Instead, report it privately via the GitHub Security Advisory tab on the affected repository,
or email us at security@purgebot.net.

Please include:

- The affected component(s)
- A description of how to reproduce the issue
- The potential impact

We will acknowledge your report within 72 hours and aim to issue a patch within 14 days.
We ask that you give us a chance to release a fix before any public disclosure.
